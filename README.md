# Simple Calculator in C

A basic calculator program written in C for practicing programming fundamentals.

## Description

This calculator can perform basic arithmetic operations including:
- Addition
- Subtraction
- Multiplication
- Division

## How to Compile

To compile the program, use:
```bash
gcc SimpleCalculator.c -o calculator
```

## How to Run

After compiling, run the program with:
```bash
./calculator
```

## Usage

Follow the on-screen prompts to:
1. Choose an operation (+, -, *, /)
2. Enter the first number
3. Enter the second number
4. View the result

## Example
```
Enter operation (+, -, *, /): +
Enter first number: 10
Enter second number: 5
Result: 15
```
